# Audiobook Production Queue

## Priority Assessment Criteria

| Factor | Weight | Description |
|--------|--------|-------------|
| **Market Demand** | 30% | Search volume, existing audiobook competition |
| **Production Ease** | 25% | Clean text availability, length, dialogue ratio |
| **Legal Clarity** | 20% | Clear PD status, no trademark issues |
| **Voice Fit** | 15% | Matches available voice profiles |
| **Revenue Potential** | 10% | Price point, audience size |
