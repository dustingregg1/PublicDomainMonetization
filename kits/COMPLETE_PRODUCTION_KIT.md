## Quick Start: Title #1 in 5 Steps

### Step 1: Get Source Text (January 1)
```bash
# Navigate to Internet Archive
# Download "The Maltese Falcon" 1930 Knopf edition
# Save as maltese_falcon_raw.txt

# Run cleanup script
python scripts/cleanup_text.py maltese_falcon_raw.txt maltese_falcon_clean.txt
```

### Step 2: Generate Content (Days 1-2)
```
1. Open PREFILLED_PROMPTS.md
2. Copy "MALTESE FALCON — Introduction Prompt"
3. Paste into Claude/ChatGPT
4. Save output
5. Repeat for all sections:
   - Character Guide
   - Historical Context
   - Discussion Questions
```

### Step 3: Quality Assurance (Day 3)
```
- [ ] Verify ALL quotes exist in the source text
- [ ] Fact-check all dates (especially author biography)
- [ ] Verify chapter counts and titles match source
- [ ] Check for spoilers in early chapter summaries
- [ ] Proofread for tone consistency
```

### Step 4: Format & Design (Days 4-5)
```
1. Assemble manuscript:
   - Front matter (from template)
   - Novel text
   - Companion content
   - Back matter (from template)

2. Format in Google Docs or Atticus

3. Create cover in Canva:
   - Art Deco noir style
   - NO film imagery
   - NO Bogart likeness
   - Include "(Annotated)" clearly
```

### Step 5: Publish (Days 6-7)
```
1. Open KDP_LISTINGS_READY.md
2. Copy all metadata for Maltese Falcon
3. Upload to KDP
4. Preview ebook and paperback
5. Publish
6. Save screenshots to Evidence folder
```

## Risk Mitigation Checklist

### Before Every Publication
- [ ] PD Dossier complete
- [ ] USPTO trademark search documented
- [ ] Source edition verified (1930, not later revision)
- [ ] Evidence screenshots saved
- [ ] Differentiation Proof created
- [ ] Dispute response template ready

### If KDP Flags Your Book
1. Don't panic - this happens
2. Open Differentiation Proof document
3. Customize response template with your ASIN
4. Attach evidence packet
5. Submit appeal within 48 hours
6. Track resolution

### Common Issues & Solutions
| Issue | Solution |
|-------|----------|
| "Insufficient differentiation" | Point to word count breakdown and originality statement |
| "Duplicate content" | Show your unique companion content |
| Trademark complaint | Use nominative fair use response template |
| Slow sales | Check keywords, adjust pricing, request reviews |
